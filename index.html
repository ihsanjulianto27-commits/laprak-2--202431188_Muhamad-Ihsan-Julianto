Soal 1: Mengapa menggunakan CSS dan JavaScript dalam pemrograman web modern?

CSS digunakan untuk mengatur presentasi (tampilan dan gaya visual) sebuah halaman web, seperti warna, ukuran teks, tata letak, dan animasi. Dengan CSS, tampilan dapat disesuaikan agar lebih menarik, konsisten, serta responsif di berbagai perangkat. JavaScript, di sisi lain, digunakan untuk menambahkan interaktivitas dan perilaku dinamis pada halaman web. JavaScript memungkinkan halaman web merespons tindakan pengguna seperti klik, input data, atau perpindahan halaman tanpa perlu memuat ulang seluruh laman.

Soal 2: Jelaskan konsep dan tujuan utama dari Responsive Web Design (RWD)!

Responsive Web Design (RWD) adalah membuat situs web beradaptasi secara mulus dengan berbagai ukuran layar (desktop, tablet, smartphone) menggunakan teknik seperti *media queries*, *flexible grids*, dan *flexible images*. Dengan menerapkan RWD, elemen-elemen pada halaman web seperti teks, tombol, dan gambar akan secara otomatis menyesuaikan posisi, ukuran, dan proporsinya agar tetap mudah dibaca serta digunakan di perangkat apa pun. Tujuan utamanya adalah memberikan pengalaman pengguna (user experience) yang optimal, konsisten, dan efisien, baik pada layar besar maupun kecil, serta meningkatkan aksesibilitas dan kinerja situs web secara keseluruhan.

Soal 3: Kombinasi CSS, JavaScript, dan RWD

Ketiganya saling melengkapi: HTML menyediakan struktur; CSS + RWD memastikan struktur tersebut memiliki tampilan yang adaptif dan rapi di berbagai ukuran layar melalui *media queries* dan tata letak fleksibel (Flexbox/Grid); JavaScript menambahkan interaksi dinamis (seperti fitur sembunyikan/tampilkan konten ini dan pengubahan mode) yang tetap berfungsi di lingkungan responsif yang diciptakan oleh RWD.
